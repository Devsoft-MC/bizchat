# BizChat Documentation

This folder contains the main planning and technical design documents for the BizChat project.

## Documents
- [sdlc.md](sdlc.md) — planned Software Development Life Cycle
- [db-structure.md](db-structure.md) — database design, including groups, conversations, and future call support

## Summary
- The project is being designed for multi-company usage from the start
- Messaging is the initial MVP focus
- Group creation, department limits, and future voice/video support are already considered in the schema planning
